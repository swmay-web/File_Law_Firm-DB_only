# A Law Firm Database

A database system designed for a law firm to manage:

- Customers
- Company clients
- Lawyers
- Cases
- Documents
- Reminders
- Payments

## Features

- Customer management
- Case tracking
- Document management
- Payment records
- Reminder system

## Technologies

- Oracle SQL
- Relational Database Design
- ERD Design

## Tables

1. Customer
2. Lawyer
3. CaseType
4. Cases
5. CustomerCase
6. Document
7. Reminder
8. Payment
